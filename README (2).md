# Global Supermarket Sales Dashboard

## 📌 Project Overview
This project is an interactive Tableau dashboard developed to analyze supermarket sales performance across multiple regions, categories, and time periods. The dashboard helps visualize business KPIs, identify profitable products, and monitor sales trends using data visualization techniques.

---

# 📊 Dashboard Features

## 🔹 KPI Dashboard
Displays important business metrics such as:
- Total Sales
- Total Profit
- Quantity Sold
- Average Sales

## 🔹 Monthly Sales Trend
Analyzes sales performance over time to identify:
- Seasonal trends
- Growth patterns
- Peak sales periods

## 🔹 Region-wise Analysis
Shows:
- Profit by region
- Sales distribution
- High-performing areas

## 🔹 Product Performance
Provides insights into:
- Best-selling products
- Product category contribution
- Revenue generation
